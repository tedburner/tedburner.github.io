Java8 Streamзҡ„д»Ӣз»Қ

VerГ¶ffentlicht am 2017-12-20 Bearbeitet am 2021-01-02 in Java Aufrufe:

еЈ°жҳҺпјҡжң¬ж–ҮдҪҝз”ЁJDK1.8

Java8жҸҗдҫӣдәҶдё°еҜҢзҡ„lambdaзҡ„APIгҖӮд»ҠеӨ©жҲ‘д»¬жқҘи®Іи®Іе…ідәҺе…¶дёӯзҡ„streamпјҲжөҒзҡ„ж“ҚдҪңпјүгҖӮеҜ№дәҺListйӣҶеҗҲпјҢжңүдәҶstreamзҡ„жөҒж“ҚдҪңпјҢж„ҹи§үеҰӮиҷҺж·»зҝјгҖӮ

з”ҹжҲҗдёҖдёӘList

дёҠйқўзҡ„д»Јз ҒStream.of дёәжҲ‘д»¬з”ҹжҲҗдәҶдёҖдёӘListпјҢдҪҶжҳҜжҲ‘д»¬йңҖиҰҒзҡ„IntegerпјҢжүҖд»ҘжҲ‘д»¬иҝҳйңҖиҰҒдҪҝз”ЁmapжқҘиҪ¬жҚўзұ»еһӢгҖӮз„¶еҗҺеҜ№дәҺз”ҹжҲҗзҡ„еҲ—иЎЁпјҢжҲ‘д»¬дёҚжғіиҰҒе…¶дёӯзҡ„3пјҢдәҺжҳҜжҲ‘д»¬дҪҝз”ЁfilterжқҘиҝҮж»ӨжҺүеҲ—иЎЁдёӯзҡ„3гҖӮеҜ№дәҺзҺ°еңЁеҲ—иЎЁжҲ‘д»¬еҸҲжғіжҳҜеҖ’еәҸеҗҺзҡ„еҲ—иЎЁзҡ„еүҚеӣӣжқЎж•°жҚ®пјҢдәҺжҳҜжҲ‘д»¬еңЁдҪҝз”ЁsortedжқҘеҜ№еҲ—иЎЁжқҘиҝӣиЎҢеҖ’еәҸеҗҺпјҢеҶҚдҪҝз”ЁlimitеҸ–еүҚеӣӣжқЎгҖӮйЎәдҫҝиҜҙдёҖеҸҘпјҢеҜ№дәҺжҜ”иҫғпјҢжҺЁиҚҗдҪҝз”ЁObjects.equals(a,b).
дёӢйқўжқҘз®ҖеҚ•зҡ„д»Ӣз»ҚдёӢstreamгҖӮ

map

mapпјҡеҜ№дәҺStreamдёӯеҢ…еҗ«зҡ„е…ғзҙ дҪҝз”Ёз»ҷе®ҡзҡ„иҪ¬жҚўеҮҪж•°иҝӣиЎҢиҪ¬жҚўж“ҚдҪңпјҢж–°з”ҹжҲҗзҡ„StreamеҸӘеҢ…еҗ«иҪ¬жҚўз”ҹжҲҗзҡ„е…ғзҙ 
mapж–№жі•жҳҜеҜ№еҲ—иЎЁйҮҢйқўзҡ„еҜ№иұЎзҡ„иҪ¬жҚўпјҢжҜ”еҰӮдёҠйқўзҡ„mapзҡ„еҠҹиғҪжҳҜеҗ§StringиҪ¬жҚўжҲҗIntegerгҖӮйҷӨдәҶиҝҷж ·зҡ„иҪ¬жҚўпјҢдҪ д№ҹеҸҜд»ҘеҜ№еҜ№иұЎиҝӣиЎҢдҪ йңҖиҰҒзҡ„ж“ҚдҪңгҖӮжҜ”еҰӮзҺ°еңЁжңүдёҖдёӘеҜ№иұЎListDTO:

зҺ°еңЁиҝҷд№ҲдёҖдёӘListпјҢдҪ зҺ°еңЁеҸӘжғіеҫ—еҲ°иҝҷдёӘеҲ—иЎЁдёӯListDTOеҜ№иұЎйҮҢйқўзҡ„idпјҢзҺ°еңЁдҪ е°ұеҸҜд»Ҙиҝҷж ·ж“ҚдҪңпјҡ

еҰӮжһңдҪ жғіиҰҒеҫ—еҲ°nameзҡ„еҲ—иЎЁпјҢд№ҹеҸҜд»ҘиҝӣиЎҢдёҠйқўзұ»дјјзҡ„ж“ҚдҪңгҖӮдёӢйқўеҜ№ListеҲ—иЎЁзҡ„е…ғзҙ е…ҲиҪ¬жҚўжҲҗеӨ§еҶҷпјҢеңЁз”ЁйҖ—еҸ·жҠҠе…ғзҙ жӢјжҺҘиө·жқҘгҖӮ

иҫ“еҮәпјҡ

flatMap

зұ»дјјдәҺmapпјҢдҪҶжҳҜflatMapжҳҜеҜ№Streamд№Ӣй—ҙзҡ„иҪ¬жҚўгҖӮ

count

count()жҳҜеҜ№еҲ—иЎЁзҡ„иҒҡеҗҲпјҢз»ҹи®ЎеҲ—иЎЁзҡ„ж•°йҮҸгҖӮ

filter

еҜ№дәҺStreamдёӯеҢ…еҗ«зҡ„е…ғзҙ дҪҝз”Ёз»ҷе®ҡзҡ„иҝҮж»ӨеҮҪж•°иҝӣиЎҢиҝҮж»Өж“ҚдҪңпјҢж–°з”ҹжҲҗзҡ„StreamеҸӘеҢ…еҗ«з¬ҰеҗҲжқЎд»¶зҡ„е…ғзҙ 

distinct

distinctжҳҜеҜ№streamйҮҢйқўзҡ„е…ғзҙ иҝӣиЎҢеҺ»йҮҚпјҢжңүзӮ№зұ»дјјдәҺSQLйҮҢйқўзҡ„distinctеҺ»йҮҚ

limit

еҜ№дёҖдёӘStreamиҝӣиЎҢжҲӘж–ӯж“ҚдҪңпјҢиҺ·еҸ–е…¶еүҚNдёӘе…ғзҙ пјҢеҰӮжһңеҺҹStreamдёӯеҢ…еҗ«зҡ„е…ғзҙ дёӘж•°е°ҸдәҺNпјҢйӮЈе°ұиҺ·еҸ–е…¶жүҖжңүзҡ„е…ғзҙ 

stream з”ҹжҲҗ map

дёҠйқўжҲ‘д»¬д»Ӣз»ҚдәҶstreamйҮҢйқўзҡ„toList()пјҢдёӢйқўжҲ‘д»¬жқҘд»Ӣз»ҚдёӢеҰӮдҪ•дҪҝз”ЁstreamзӣҙжҺҘз”ҹжҲҗmapпјҢиҖҢдёҚжҳҜеғҸд»ҘеүҚдёҖж ·пјҢйңҖиҰҒеҫӘзҺҜеҗҺеҶҚдёҖдёӘдёҖдёӘзҡ„putиҝӣmapгҖӮиҷҪз„¶жҲ‘д»¬д№ҹеҸҜд»Ҙз¬¬дёүж–№зҡ„jarз”ҹжҲҗMapпјҢжҜ”еҰӮGoogleзҡ„жҹҗдәӣjarпјҢдҪҝз”ЁйҮҢйқўзҡ„Maps.uniqueIndex()д№ҹеҸҜд»ҘзӣҙжҺҘжҠҠListиҪ¬жҚўжҲҗMapгҖӮ

Instant

Instant代表某个时间，类似于java.util.Date，它是精确到纳秒的（而不是象旧版本的Date精确到毫秒）。如果使用Long去表示，其中的存储空间是不够的，不能像data那样直接使用一个long类型表示时间戳。其实在Instant中，它的源码就是分成两部分组成的。
